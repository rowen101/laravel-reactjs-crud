

 ## Installation

### Clone repo

``` bash
# clone the repo
$ git clone https://github.com/rowen101/laravel-reactjs-crud.git
```

### Copy and Paste

Copy all your files to your project folder and then,

``` bash
# go into app's directory
$ cd my-project

# install app's dependencies
$ npm install

# install laravel dependencies
$ composer install

# run migration
$ php artisan migrate
```

## Usage

``` bash
# run laravel
php artisan serve

# watch dependencies
npm run watch
```
